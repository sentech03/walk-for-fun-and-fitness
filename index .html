Beginner Guide

  1. Start with 10–20 minute easy walks, 3–4x/week.
  2. Wear comfy shoes and moisture‑wicking layers.
  3. Bring water; use sunscreen/hat if sunny.
  4. Pick flat, safe routes; add hills gradually.
  5. Warm‑up 5 min; cool‑down and light stretching.
  6. Track steps or time; increase ~10% per week.

Tip

If you feel pain (not normal effort), slow down, shorten duration, or rest. Build consistency first.
